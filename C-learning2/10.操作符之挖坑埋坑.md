    //代码5
    #include <stdio.h>
    int main()
    {
        int i = 1;
        int ret = (++i) + (++i) + (++i);
        printf("%d\n", ret);
        printf("%d\n", i);
        return 0;
    }

Linux: 10 4
visual studio: 12 4

这段代码中的第—个+执行的时候，第三个++是否执行，这个是不确定的，因为依靠操作符的优先级和结合性是无法决定第一个+和第三个前置++的先后顺序。
